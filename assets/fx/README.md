# FX pixel-art à ajouter (fond transparent)

Le jeu tente de charger automatiquement des images d'effet avec ce schéma:

`./assets/fx/<type>_<groupe>.png`

- Types: `feu`, `eau`, `plante`, `acier`, `vol`, `poison`, `sol`, `glace`, `psy`, `tenebres`, `combat`, `normal`
- Groupes: `offensive`, `terrain`, `renforcement`, `controle`

Si l'image n'existe pas, le jeu garde un fallback visuel (dégradés/filtres).

## Conseils de style

- PNG 128x128 ou 256x256, fond transparent.
- Contraste fort, 3 à 6 couleurs max.
- Préférer des contours nets pixel-art (pas de blur peint).
