Kawaiimon - version locale corrigée

Ouvre index.html directement dans ton navigateur.
Cette version utilise js/app.bundle.js pour fonctionner hors ligne en double-cliquant sur index.html.

Dossier :
- index.html
- css/styles.css
- js/app.bundle.js   -> version runtime locale
- src_js/*.js        -> fichiers source séparés pour modifier le projet
- assets/...         -> images et fond d'arène


Musique de fond :
- Place le fichier `pokeitan final.wav` à la racine du projet, au même niveau que `index.html`.
- La page tente de lancer automatiquement la musique en boucle dès l'ouverture.
- Selon le navigateur, l'autoplay sonore peut être bloqué jusqu'à la première interaction clavier / souris / tactile.
